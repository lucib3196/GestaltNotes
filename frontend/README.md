# Installation

Minimal frontend install:

1. Create `frontend/.env` with:
```
VITE_LANGSMITH_API_KEY=
VITE_PRODUCTION_URL=
VITE_LOCAL_URL=http://127.0.0.1:2024
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_PROJECT_ID=
VITE_STORAGE_BUCKET=
VITE_MESSAGING_SENDER_ID=
appId=
```

2. Install dependencies:
```
npm install
```

3. Start the dev server:
```
npm run dev
```
